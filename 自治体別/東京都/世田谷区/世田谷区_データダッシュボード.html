標準地域コード 13112東京都e-Stat API / 社人研 / 国勢2025速報

概要・ハイライト

構成は、世田谷区基本計画（2024-2031）の9分野22の分野別政策（子ども・若者／教育／健康・福祉／災害・危機管理／環境・リサイクル・みどり／経済・産業／文化・スポーツ／都市整備／人権・コミュニティ）を、人口＋6分野＋行財政の8分野へ整理した形。各分野タブで「担当課／計画の成果指標（現状→目標）／統計データ」を束ねています。成果指標は重点政策・分野別政策の〈現状（令和5年度）→中間（令和9年度）→最終目標（令和13年度）〉から抽出しています。

計画を読む

計画の変遷（基本構想2013＝九つのビジョン→前基本計画2014-2023→現行の世田谷区基本計画2024-2031）と、計画本文のテキスト・原典PDF。
世田谷区基本計画は、各政策の成果指標を〈現状（令和5年度＝2023）→中間（令和9年度）→最終目標（令和13年度＝2031）〉の形式で設定しています（前計画との連続A/B/C達成記号は計画改定により非連続）。各分野タブでは、計画の成果指標（現状→目標）と本区の統計実績を対応させて表示しています。
