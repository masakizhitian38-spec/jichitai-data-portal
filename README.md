# 全国自治体データポータル

全国1,741市区町村の公開統計を、1ページで横断的に確認できるデータポータルです。総合計画・長期計画づくりの現状把握や、近隣・類似団体との比較の出発点としてお使いください。

**公開URL:** https://masakizhitian38-spec.github.io/jichitai-data-portal/

## できること
- トップの**地方区分**から都道府県 → 市区町村を選ぶと、人口・経済・産業・労働・教育・医療福祉・財政などを**分野別タブ**で表示
- 主要指標は2010年以降の**経年グラフ**（本市・県平均・全国平均の3本／指数=基準年100で伸び率を比較）
- 人口は2000〜2050年（実績＋将来推計）、自然増減・社会増減の推移
- **県内順位**・全国/県平均との比較、特徴が近い「**似ている自治体**」の自動抽出
- 指標名にカーソルを置くと**定義と評価の向き**を表示

## 全市区町村について
すべての市区町村は `index.html` に内蔵されており、画面の操作のほか、URL末尾にコードを付けて直接開けます。
例）狭山市: `…/jichitai-data-portal/#11215`

## データ出典
総務省 e-Stat（国勢調査・社会人口統計体系）、国立社会保障・人口問題研究所（地域別将来推計人口）ほか。各ページ下部の「出典」に明記しています。数値は取得時点のもので、確報・最新値は各統計の原典をご確認ください。計画の成果指標・達成状況などの「計画層」は含みません。

## 構成
- `index.html` … サイト本体（全1,741市区町村のデータを内蔵）
- `自治体別/…` … 一部自治体の詳細版ダッシュボード
- `.nojekyll` … GitHub Pages 用の設定ファイル（削除しないでください）

本サイトは公開統計から自動生成しています。お気づきの点はサイト内のアンケートからお寄せください。
